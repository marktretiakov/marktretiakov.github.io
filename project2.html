Distribution Center Robot

I developed an autonomous delivery robot using LEGO Mindstorms EV3, designed to operate efficiently and autonomously in fulfillment environments. The robot is capable of loading, sorting, and delivering packages with minimal human intervention. It uses a color sensor to identify and sort packages into separate compartments, while a lifting mechanism ensures accurate placement onto the robot. The loading phase is completed autonomously, with the robot returning to its starting position after each package is secured.

The robot employs advanced routing techniques, including Proportional Derivative (PD) line-following, using two color sensors to navigate accurately along predefined paths. Upon reaching designated offloading zones, the robot detects color-coded markers and automatically executes delivery. Built with error-handling features, including an emergency stop sensor and user input options for repeating or shutting down operations, the robot adapts to various conditions. Originally conceived as a basic line-following machine, the project evolved into a fully autonomous system that showcases innovation in robotics, sensor integration, and autonomous delivery systems.
